What Is This?
-------------

This is an example of a web application developed in ASP.NET C # in order to integrate with facebook using the Facebook Graph API tool.

Currently this version automatically searches for information from friends, groups, tastings and posts.


How To Use The Examples
-----------------------

The ways to interact with the examples in this project:

1. You have to be connected with Facebook

2. Run the solution application.

3. Accept the permissions required to the facebook application to access from your account.
